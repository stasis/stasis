Documentation

duplicate a buffer, similar to strdup

Parameters:
tag - tag used for memory leak detection
data - buffer to duplicate
size - size of that buffer
Returns:
void * retptr == NULL: OUT OF MEMORY
void * retptr != NULL: pointer to buffer allocated
